# Prototypes

Isolated throwaway prototypes for Gene Forge. Each prototype lives in its own subdirectory and is independent from the main `Assets/` source tree.

## Rules
- Each prototype MUST have its own README explaining what it tests
- Prototypes are NOT production code — they validate ideas
- Delete or archive prototypes once the idea is validated or rejected
- Never import prototype code directly into `Assets/Scripts/`

## Planned Prototypes
- Grid rendering (GPU instancing vs individual tiles)
- Isometric camera rig (Cinemachine setup)
- Creature DNA UI (splice/graft interface mockup)
- A* pathfinding with height costs
