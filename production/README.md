# Production

Sprint plans, milestone definitions, and release tracking for Gene Forge.

## Structure
```
production/
├── sprints/        # Sprint plan documents
├── milestones/     # Milestone definitions and criteria
└── releases/       # Release notes and checklists
```

## Milestones
1. **MVP** — 27 core systems, playable combat loop
2. **Vertical Slice** — Full encounter + DNA alteration + campaign segment
3. **Alpha** — All systems integrated, content pipeline flowing
4. **Full Vision** — All 53 systems, polish, post-MVP features
